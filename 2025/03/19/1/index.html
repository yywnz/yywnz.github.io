云计算技术 课程实验报告
 
学号：202200130182 姓名：张晓泉 班级： 22.2
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 2025/3/12
实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。
具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 
联网的计算机一台

软件环境：
Windows or Linux

实验步骤与内容：
1.创建github仓库

2.安装node.js和git
3.创建存放仓库文件的目录

4.选中创建的目录，右键打开git窗口，进行hexo的安装，然后init初始化

使用npm install hexo-server –save命令安装hexo的服务器，这样可以使用npx hexo server启动服务器

设置ssh,在github中设置ssh密钥，与github链接

成功上传到github

之后使用npx hexo g和npx hexo d来重新生成文件并上传至github

结论分析与体会：
通过这个实验，我对于使用github和hexo结合来部署自己的博客网站有了更深的认识，可以熟练操作、搭建、添加、编辑博客。在实验的最后我也成功创建且编辑了一篇博客，并上传到github，虽然在实验中我遇到了诸如npm无法操作成功使用，直接使用hexo无法成功的罕见问题，但是我仍通过上网搜索、不断尝试来解决了这些问题。例如经过原因查询，在hexo前面加上npx就能成功的原因是：npx 想要解决的主要问题，就是调用项目内部安装的模块，所以可以理解为在命令行下调用，可以让项目内部安装的模块用起来更方便，npx运行的时候，会到node_modules/.bin路径和环境变量$PATH里面，检查命令是否存在，所以系统命令也可以调用，即上面的直接hexo init命令安装不成功的时候加上npx的话也许就可以成功了。并且经过实验，我认为使用github和hexo来搭建博客可以让博客的文件保持在本地，无需网路就能打开，极大的方便了用户。
